Use the following ordered and unordered lists.

  Ordered List
  ***************************************

  #. Step 1.
  #. Step 2.
  #. Step 3.

  Unordered List
  ***************************************

  * Item 1.
  * Item 2.
  * Item 3.


  Unordered List Inside Ordered List
  ***************************************

  #. Step 1.

      * Item 1.

      * Item 2.

  #. Step 2.

  Ordered List Inside Unordered List
  ***************************************

  * Item 1.

    #. Step 1.

    #. Step 2.

  * Item 2.


  Unordered List Inside Unordered List
  ***************************************

  * Item 1.

    * Item a.

    * Item b.

  * Item 2.

  Ordered List inside Ordered List
  ***************************************

  #. Step 1.

      #. Step a.

      #. Step b.

  #. Step 2.

  Code, Images, and Other Content Inside of Lists
  *************************************************

  #. Step 1. Example:

      .. code-block:: bash

        Example code

  #. Step 2.
